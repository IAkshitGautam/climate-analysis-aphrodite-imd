# climate-analysis-aphrodite-imd

This repository contains R scripts used for:
Spatio-temporal analysis of precipitation and temperature in Madhya Pradesh (1981–2015) using APHRODITE data and validation with IMD datasets.

Script 1: APHRODITE processing and analysis  
Script 2: IMD comparison and validation
